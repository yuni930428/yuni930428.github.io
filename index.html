期末專案：營養推薦系統

在這個期末專案中，我開發了一個基於人工智慧的個性化營養推薦系統。該系統能根據使用者的健康資料、飲食偏好以及目標（如減重、增肌、保持健康）提供量身定制的營養計劃。

技術背景：

該系統利用機器學習模型分析使用者的飲食記錄和健康數據，並根據預設的營養標準進行推薦。我使用了 Python 和 Flask 架設前後端，並且利用 TensorFlow 來進行預測。

主要功能：

  • 根據健康資料提供營養建議
  • 追蹤並記錄飲食偏好
  • 提供每日餐單推薦，適應不同目標需求
